Text Explainer

How to use:

  1. Enter your Gemini API key above
  2. Select text on any webpage
  3. Use one of these methods:
    • Double-click the text
    • Press Alt+E after selecting text
    • Press Tab key after selecting text
    • Click the "Explain" button that appears
  4. View the AI-powered explanation!
